About this Project.

In this project I used a Raspberry Pi Pico W to control a forklift that was 3D printed using both FDM and Resin printing. The prepose of this forklift is to pull a cart around that itself can have different functions. The project is still a work in progress so in the next part of this project I will be adding better speed control and arm movement. Right know the arm goes from one position to the next no smoothing over which can be entertaining to watch as it flings object across the room (Sadly thatâ€™s not what I want this to do.). When this is done, I will start on the trailer for it this trailer should be able to hold anything that I might want to pull along. One idea for this is to use it to garden by changing the forks out for s different tool and the cart carrying seeds. I will be able to plant using this method. The method of power to the Pico is from two AA batterys but this capacity can be increased with the cart if that is needed but on the 2 AA batterys it can drive in circles for around 2-5 hours depends on how many times you life the forks.

Things used in the project include:
  1. Computer
  2. Raspberry Pi Pico Wireless
  3. 3D Printer
  4. X2 geared Motors
  5. X2 Servos
  6. X2 AA Battery's
  7. Wires
  8. Solder
  9. Soldering Iron
  10. Spring
  11. Time to kill
